extension UITraitCollection
  • hasDifferentBreakpoint(comparedTo:)

    Return whether this trait collection, compared to a different trait collection, could represent a different breakpoint.

    Breakpoints are design specific, but a common one is at 600 points. Ideally looking at window size would be the way to detect every breakpoint change.

    If you need to be aware of when breakpoints might change, override traitCollectionDidChange in your view or view controller, and use this method to compare self.traitCollection with previousTraitCollection.

    Currently, a change in any of these traits could affect breakpoints: horizontalSizeClass, verticalSizeClass and more could be added in the future.

    Declaration

    Swift

    public func hasDifferentBreakpoint(comparedTo traitCollection: UITraitCollection?) -> Bool

    Parameters

    traitCollection

    A trait collection that you want to compare to the current trait collection.

    Return Value

    Returns a Boolean value indicating whether changing between the specified and current trait collections could affect breakpoints.

  • hasDifferentFontAppearance(comparedTo:)

    Return whether this trait collection, compared to a different trait collection, could show a different appearance for fonts.

    If you need to be aware of when fonts might change, override traitCollectionDidChange in your view or view controller, and use this method to compare self.traitCollection with previousTraitCollection.

    Currently, a change in any of these traits could affect fonts: preferredContentSizeCategory, legibilityWeight and more could be added in the future.
